# Pre-ship / release-readiness (starter seed)

**When this `starter_repo/` folder still lives inside Project Forge:** use the **canonical** checklist at [`../../docs/RUNBOOK_PreShipChecklist.md`](../../docs/RUNBOOK_PreShipChecklist.md) — same **required / conditional / N/A-with-reason** structure; [`docs/ai/AGENT_STARTUP_CHECKLIST.md`](ai/AGENT_STARTUP_CHECKLIST.md) defines when to use it and ties verification notes to that canonical checklist.

**After you copy this seed into a new repository:** copy that canonical runbook into **`docs/RUNBOOK_PreShipChecklist.md`** in your new repo (replace this file) so the project owns a **single** pre-ship source of truth, or author an equivalent checklist **with the same structure** (required vs conditional vs N/A policy). Do not let KB exports or chat become the checklist of record.
