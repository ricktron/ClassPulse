# Start here

This folder is a **starter** for a new repo.

**Do this first**

1. Fill out **`STARTER_CARD.md`**. Pick one letter per question, or write **not sure yet**.
2. If you are using AI, paste **`START_HERE_FOR_LLM.md`** and your filled starter card into your first message.
3. Start working. Use **`README.md`** when you need orientation.

You do **not** need to read the whole repo before your first real edit. Deeper guidance is here when you need it.
